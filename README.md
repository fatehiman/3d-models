# 3D Models for 3D Printing — Free STL Files (MIT Licensed)

A free, open-source collection of **3D printable models** in **STL**, **3MF**, and Blender (**.blend**) source formats. Every model in this repository is released under the permissive **[MIT License](LICENSE)** — free to download, print, modify, remix, and use commercially.

Models are designed and sliced for the **Creality Ender-3** (most include ready-to-print `.gcode`), but the STL files print on any FDM 3D printer.

## 🖨️ How to Use

1. **Download** the `.stl` (or `.3mf`) file for the model you want.
2. **Slice** it in your slicer (Cura, PrusaSlicer, etc.). Ender-3 users can often use the included `.gcode` directly.
3. **Print** on any FDM 3D printer.
4. **Edit** — open the `.blend` source files in [Blender](https://www.blender.org/) to customize a model.

### File Formats
- **`.stl`** — ready-to-slice mesh (universal).
- **`.3mf`** — modern mesh format with print metadata.
- **`.blend`** — editable [Blender](https://www.blender.org/) source.
- **`.gcode`** — pre-sliced for Creality Ender-3.
